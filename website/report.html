ArgoCD

ArgoCD is an app used to create, scale and monitor other apps as well as everything connected to them inside a cluster in an orderly manner. Argo Rollouts help automate and manage update releases of your applications.

Renovatebot

De renovate bot is a tool used to monitor dependancies and create pull requests when those have updates available and automatically updates them. Ours runs in silent mode, meaning it doesn't work automatically and we have to run it manually every now and then.
